- **Need for multi-core CPUs?**
  - Run multiple execution threads concurrently.
  - Applications can be high performance (high bandwidth) or low performance.

- **OS's Scheduler**
  - Which processor should run which thread?
  - The kernel of the OS has the scheduler, the primary decision-maker for task execution.
  - The scheduler uses different algorithms to determine the order and placement of tasks on available processing units.
